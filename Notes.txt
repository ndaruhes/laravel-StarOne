B. ARTISAN COMMAND
==============
1. php artisan serve
	- menjalankan project

2. php artisan make:controller
	- membuat controller baru

3. php artisan make:model
	- membuat model baru

4. php artisan make:migration
	- membuat migration baru

5. php artisan migrate
	- menjalankan migrasi database

C. DIRECTORY
==============
1. Routing Directory	=> routes
2. Templating Directory	=> resources->views
3. Controllers		=> app->Http->Controllers
4. Models		=> app->Models
5. Migration		=> database->migrations
